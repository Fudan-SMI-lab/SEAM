SpeechGPT-2.0-preview migrated smoothly as an ordinary_cuda project with no custom CUDA operators. Phase 4 found only 1 rule match (torch_npu import injection in transformer.py). The primary challenge was the Gradio web UI entry point blocking automated validation, resolved by adding a --headless CLI flag gating the UI and providing NPU validation fallback. A secondary fix added graceful auto-fallback when the model checkpoint directory is absent — a common scenario when large (~14GB) HuggingFace model weights are not downloaded in the validation environment. The migration produced 3 Phase 5 iterations with final exit_code=0, NPU device detection confirmed, and core project imports verified.
