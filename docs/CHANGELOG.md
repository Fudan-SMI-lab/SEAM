# Changelog

## [v1.1.0] - 2026-05-27
### Added
- SEAM新增支持2款GPU的迁移适配。
- 异构GPU环境自识别，和自动选配YAML文件运行。
- 支持基于用户当前镜像环境、或自动选择用户本地镜像创建容器，进行增量适配，提升迁移效率。

### Changed
- 调整用户接口和文档，使更加易用。
- 仓库重构，优化代码结构，提高可维护性。

### Performance
- 优化执行逻辑，整体运行性能提升。


## [v1.0.0] - 2026-04-30
### Added
- SEAM的仓库主体完成，实现3款框架模型在1款GPU上实现自动化迁移。
