# 阿里平头哥PPU

PPU（Parallel Processing Unit）是阿里巴巴平头哥半导体推出的训推一体 AI 加速芯片，是大模型训推一体的高性能GPU设备，生态易用性较好。

## 值得关注的型号：

1. 平头哥 真武 810E

- 显存：96GB HBM2e
- 片间互联：700 GB/s（自研），支持万卡集群
- 接口：PCIe 5.0
- 单卡功耗：≤ 400W

整机细节可以参考阿里云上AI Stack产品说明 ：https://www.aliyun.com/product/ai-stack/pricing

根据公开信息，官方型号为真武 810E。真武810E的研发工作于2020年启动，并于2022年底至2023年初完成了研发和场景验证 。2025年初，该芯片开始量产。2025年11月，阿里巴巴开始向企业客户销售搭载该芯片的AI Stack大模型训推软硬一体解决方案。2026年1月29日，阿里平头哥官网上线高端AI芯片“真武810E”。真武810E在阿里云有万卡集群部署案例。

2. 平头哥 真武 M890

- 2026年05月20日发布，https://www.t-head.cn/news/newsDetail?id=188 。SEAM团队还没有用上
- 显存：144GB HBM（比 810E 多 50%）
- 片间互联：800GB/s（+14%）
- 性能：官方明确 ≈ 810E 的 3 倍
- 精度：原生支持 FP32 ~ FP4，全场景覆盖
- 集群：128 卡超节点，百纳秒级互联
- 定位：面向 大模型训练 + 超高并发推理（Agent 时代）
