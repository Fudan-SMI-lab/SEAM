# 沐曦 MetaX GPU

沐曦（MetaX）成立于 2020 年，2025 年 12 月在科创板上市，估值很高，是国内少数实现全栈自研、大规模商用、千卡集群落地的通用 GPU 公司。

产品覆盖：
- 曦云 C 系列：AI 训练 + 通用计算 GPU，如曦云 C500 / C550 / C600，CUDA适配。
- 曦思 N 系列：AI 推理 GPU
- 曦彩 G 系列：图形渲染 GPU
- 曦索 X 系列：科学计算加速卡

## SEAM中说的型号

1. 曦云C550 
- 显存：64GB HBM2e，ECC保护
- 互联：8卡全互联，带宽896GB/s，支持 2–64卡超节点
- 单卡功耗：350–450W

复旦大学参与建设的曦云C550 Shanghai Cube 液冷整机柜，坐落在复旦大学的上海张江校区，开放产教融合合作参观和使用，有需要了解的可以通过SEAM团队联系。

2. 曦云C600
- 显存：144GB HBM3e，ECC保护
- 互联：自研 MetaXLink 2.0，以及Ethernet互连，支持最大256卡超节点
- 单卡功耗：400-1000W
